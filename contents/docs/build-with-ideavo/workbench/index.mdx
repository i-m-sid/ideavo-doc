# Workbench

The Ideavo Workbench is your complete development environment in the browser.

![Workbench](/images/workbench.png)

## File Tree
- Browse and navigate your project structure
- Create, rename, and delete files and folders
- View file modifications in real-time
- Expand and collapse directories

## Code Editor
- Full-featured code editor with syntax highlighting
- IntelliSense and auto-completion
- Real-time error detection and suggestions
- Support for Next.js and React Native Expo projects

### Editor Features
- **Syntax Highlighting**: For JavaScript, TypeScript, CSS, and more
- **Code Folding**: Collapse code blocks for better navigation
- **Find & Replace**: Search and replace text across files
- **Multi-cursor Editing**: Edit multiple lines simultaneously

## Preview
- Live preview of your application
- Automatic refresh when code changes
- Mobile-responsive preview for different screen sizes

### Preview Features
- **Hot Reload**: See changes instantly without manual refresh
- **Device Simulation**: Test different screen sizes and orientations
- **Console Access**: View browser console logs and errors

## Terminal
- Built-in terminal for running commands
- Package management (npm, yarn, pnpm)
- Build and development scripts
- Environment variable management

### Terminal Commands
- `npm install` - Install dependencies
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run test` - Run tests

## File Editing Restrictions

**Important**: You can only edit files when your GitHub account is connected.

### Without GitHub Connection
- **Read-only mode**: View files and folder structure
- **No file editing**: Cannot modify, create, or delete files
- **Preview only**: Can view the live preview of your application

### With GitHub Connection
- **Full editing capabilities**: Create, modify, and delete files
- **Real-time sync**: Changes are automatically saved
- **Version control**: All changes are tracked in git
- **Collaboration**: Share your repository with team members
