# Workbench

The Ideavo Workbench is your complete development environment in the browser.

## Preview

- Live preview of your application
- Automatic refresh when code changes
- Manual reload button if needed
- Download full code using download button
- Shareable preview link

![Preview](/images/preview.png)

## Env Editor

- Shows the current environment variable configuration of application
- Add your API keys for third party integration here
- Saving the env file invokes reload of app preview

![Env Editor](/images/env-editor.png)
